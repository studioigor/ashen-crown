## Геймплей и механики

### 1. Тултипы при наведении на юнитов и здания
**Что делать:** при hover мышью по юниту/зданию показывать панель с типом, HP, атакой, радиусом обзора, способностями. Для рабочих — сколько и каких ресурсов несут (`carrying.amount`, `carrying.type`). Тултип должен прятаться при уходе курсора и не перекрывать команды.
**Где:** `src/scenes/UIScene.ts` (уже есть тултипы у кнопок через mode-text — переиспользовать стиль), `src/scenes/GameScene.ts` (hover-детект через pointermove + spatial lookup юнитов/зданий).

### 2. Автопилот
**Что делать:** опциональный тогл «AI управляет юнитом. Тот же контроллер из `AI.ts` запускается на юнита или выделенную группу юнитов; игрок может в любой момент перехватить управление выделением и приказом.

### 3. Группы юнитов: строй, обход, поддержка ≥ 2000
**Что делать:** при групповой move-команде распределять юнитов по слотам строя (`Formation.ts` уже считает позиции — подключить к issueMoveCommand), на пути обходить стоящих союзников вместо толкания, делать batched A* для группы (один расчёт + распределение).
**Где:** `src/systems/Pathfinding.ts`, `src/systems/Formation.ts`, `src/scenes/GameScene.ts` (issueMoveCommand для группы).

### 4. Стабилизация pathfinding при больших группах
**Что делать:** ограничить частоту repath по группе (cooldown), внедрить flow-field или HPA* для масс-движения, профилировать `findPath` (текущий лимит 4000 итераций — проверить, упирается ли). Добавить unit-test на сценарий «60 юнитов идут через узкий проход».
**Где:** `src/systems/Pathfinding.ts`.

### 5. Просадки FPS при росте количества юнитов
**Что делать:** замерить кадр на 100 / 300 / 500 юнитах; включить object pooling для floating-damage-text и партиклов; кешировать health bar текстуры; снизить частоту обновления FOW при большом количестве сущностей; не апдейтить визуалы юнитов вне камеры.
**Где:** `src/scenes/GameScene.ts`, `src/world/FogOfWar.ts`, `src/systems/ParticleFX.ts`, `src/systems/EffectsSystem.ts`.

### 6. Корованы (easter egg)
**Что делать:** периодически по дороге через карту проходит NPC-караван; военные юниты могут атаковать его, при уничтожении выпадает золото/дерево. Появление редкое, не ломает баланс.
**Где:** новый `src/entities/Caravan.ts`, регистрация и таймер спавна в `src/scenes/GameScene.ts`.
