# Ashen Crown

Эта игра — результат эксперимента по созданию RTS в стиле Warcraft с помощью нейросетей.

Подробности в видео на YouTube.

## Технологии

- TypeScript + Vite
- Phaser 3
- Процедурная генерация графики (спрайты, текстуры, анимации — всё рисуется кодом, внешних ассетов нет)

## Запуск

### Онлайн

https://studioigor.github.io/ashen-crown/

### Локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Сборка кладётся в `docs/` — GitHub Pages раздаёт её с ветки `main`.
